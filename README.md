## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/JoseNeto5689/bd2_project.git
    ```
2. Navigate to the project directory:
    ```bash
    cd bd2_api
    ```
3. Install the dependencies:
    ```bash
    npm install
    ```
